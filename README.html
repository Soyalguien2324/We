Abrir el viejo “Panel de control”.  
Una vez se haya abierto el viejo Panel de control, hay que pulsar sobre “Programas”.  
Pulsa en “Activar o desactivar las características de Windows”, apartado al que  
también es posible llegar tras acceder a “Programas y características”.  
Hacia abajo hasta ver “Subsistema de Windows para Linux”, marcar para instalar  
después sobre el botón Aceptar. Tras terminar la instalación de la característica, reinicia  
el pc  
Instala Ubuntu en la Microsoft Store  
Abre Ubuntu y ya está.
